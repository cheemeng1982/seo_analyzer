README SEOAnalyzer v1.0
15 April 2018

===============
Requirements
===============
(a) .NET Framework 4.5 installed
(b) IIS 6.0 or Visual Studio.NET 2015 or higher


================
Instruction
================

Please follow below steps to setup the applications.

(1) 	Ensure "listStopWord.txt" is correctly located as per the key value "StopWordListPath" (web.config) defined.
(2) 	If using IIS, setup the application in IIS Manager pointing to "SEOAnalyzer" folder. Provide necessary domain name for both.
  	For example - "http://www.seoanalyzer.com". Make sure the local machine host file for that domain is pointing to localhost.
(3)	If using VS.NET, just run the application as usual.
